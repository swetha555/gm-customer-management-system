This project is a Single page Angular web application!!
This gives the list of customers and managing the data with delete, edit options. You can add a customer by clicking on the add button. 

In order to run the app,

1. Go to terminal--> cd to the project diectory similar to "cd /Users/swetha/Downloads/gm-customer-management-system.Web"

2.  if you have node or python already installed, just run:
a) npm install - to download and install all the dev dependencies. 
b) npm start - to start the app in the posrt 8085.

3. Now go to your desired browser and type http://localserver:8085

You will see the app up and running.
